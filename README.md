# MeteorologicalAnalyses
利用Spark对气象监测数据进行基本的分析

## 目录说明

1. `./analyses` 数据分析的结果，以csv格式存放  
2. `./data` 原数据，包括预处理好的`preprocessed*.csv`  
3. `./source` 源代码
   - `/Jupyter` 数据预处理与可视化
   - `/Scala-Spark-DataAnalyze` Spark数据分析代码
